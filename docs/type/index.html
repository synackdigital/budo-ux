Typography

Budo-UX sets all type in Nimbus Novus Sans Condensed by default. This font is used for headlines and UI text of all sorts.

Body text for pages and news articles is typeset in URW Garamond.

Font files are included in dist/fonts and are linked to utility classes for easy access.

Headlines and titles

Use one of the following type classes to create a headline or title:

<span class="type-h0">Head 0</span>
<span class="type-h1">Head 1</span>
<span class="type-h2">Head 2</span>
<span class="type-h3">Head 3</span>
<span class="type-h4">Head 4</span>
<span class="type-h5">Head 5</span>
<span class="type-h6">Head 6</span>

Type areas

To format text from e.g. a wysiwyg editor, wrap the whole content inside a container with .typearea. This will set all contained text to URW Garamond, except for some html-tags like headlines and lists that are styled as if they had type classes set on them.

<div class="typearea">
  <h4>Headline</h4>
  <p>This paragraph should display in URW Garamond.</p>
</div>
